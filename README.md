## Overview
    A documentation on learning how organization relies on Microsoft Entra ID for authentication and Microsoft 365 (M365) for collaboration and email.

# Learning Objectives
    Basic understanding of how to use logs to identify attacks with Entra ID and M365 logs.
    Understand Entra ID and M365 log types and core structure.
    Understand Entra ID and M365 log as critical log sources for modern SOC investigations.
    Understand the risks of identities and why attackers target them in modern environments.

# Topics Covered
    Module 1: What are Identity Providers
    Module 2: Identities as the Target
    Module 3: Entra ID Sign-in Logs
    Module 4: Entra ID Audit Logs
    Module 5: M365 Introduction
    Module 6: M365 Audit Logs
